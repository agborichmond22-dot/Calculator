# Palette's Journal

## 2025-07-20 - Initializing Palette's Journal
**Learning:** Foundational review of the calculator UI components revealed several a11y gaps: the screen reader display lacked dynamic polite announcement attributes, and interactive buttons lacked descriptive ARIA labels and focus indicators.
**Action:** Always verify a11y elements (like aria-live, aria-label, and focus-visible css outline styles) immediately during exploration of single-page utility apps.
